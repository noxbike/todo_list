Comment utiliser cette application web :

- Vous devez tout d'abord possédez ceux ci:
    . Wamp ou Xampp
    . Php
    . Composer

- Téléchargé le fichier sur votre bureau

- Ouvrez le dossier Sql et importer le dans votre base de donnée

- Ensuite le fichier a la racine .env.example supprimer '.example' 
  et ajouter les donnée de votre base de donnée afin de pouvoir 
  se connecté sur Xampp ou Wamp

- Ensuite ouvrez le terminal(linux) ou Powershell(windows)

- Assurez vous de bien vous situez dans le dossier todo_list

- Ensuite tapez la commande php artisan serve

- Ensuite ouvrez votre navigateur et tapez dans la barre url en haut
'localhost:8000/todo'
